set(string $dotNotation, mixed $value) : $this

Places a value into a tiered array. A dot notation of "one.two.three" would place the value at ['one' => ['two' => ['three' => 'valueHere' ]]] If the array does not exist it will be added. Indexed arrays can be used, and referenced by number. i.e. "one.0" would return the first item of the one array.

since 1.0.0
